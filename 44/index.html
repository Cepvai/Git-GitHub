Git ir GitHub

Git'as - populiariausia kodo versijavimo programa.
GitHubas'as - vienas populiaresnių versijuojamo kodo talpinimo platformų.

Git komandos

  • init - sukuriama git repozitorija dabartiniame aplankale
  • status - nurodo informacija apie lokalios git'o repozitorijos statusą
  • add - prideda nurodytus failus prie commit'inimui paruoštų failų (stage'ina)
  • rm --cachrd || restore --staged - išima nurodytus failus iš stage'inimo zonos (unstage'ina)
  • commit -m 'zinute' - sukuria kodo versiją iš stage'intų failų ir prideda žinutę apie pakitimus (žinutė turetų būti trumpa (ne ilgesnė nei 50 simbolių) ir aiški)
  • restore - sugrąžina faila į buvusio commit'o versiją (panaikina visus pakitimus darytus po paskutinio commit'o)
  • git push - nusiunčia
  • git pull - atsisiunti is GitHub
  • clone - klonuoja nurodyta repozitoriją iš išsorinės repozitorijos
  • log - išveda commitų sąrašą
  • checkout commit_ID|brancName - pereina į nurodyto kodo versiją arbą kitą šaką
  • branch naujosSakosPavadinimas - sukuria nauja šaką
